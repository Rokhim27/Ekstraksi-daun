# Ekstraksi Fitur Warna Daun (HSV)

## Deskripsi
Project ini melakukan ekstraksi fitur warna pada citra daun menggunakan metode HSV.

## Tools
- Python
- OpenCV
- Google Colab

## Dataset
10 citra daun ukuran 256x256

## Output
- fitur_daun.csv
- fitur_daun.xlsx

## Kesimpulan
Metode HSV efektif untuk analisis warna daun.
